
# XSSFind

Python Kullanılarak yapılmış XSS Finder.
## Kullanılan Teknolojiler

**Dil:** Python

**Modüller:** Colorama , Requests

  
## Bilgisayarınızda Çalıştırın

Projeyi klonlayın

```bash
  git clone https://github.com/furkzsec/XSSFind
```

Proje dizinine gidin

```bash
  cd XSSFind
```

Kodu çalıştırın

```bash
  python3 xss.py 
```
