# XSSFind

XSS Finder built using Python.
## Technologies Used

**Language:** Python

**Modules:** Colorama, Requests  
## Run on Your Computer

Clone the project

```bash
  git clone https://github.com/furkzsec/XSSFind
```


### Navigate to the project directory

```cd XSSFind```

### Run the code

```python3 xss.py```
